# 🎯 1984 | Global & Competitive Bot

Telegram-бот, развивающийся в рамках проекта "1984 Bounties Competitive".

> *"Вы больше не участник — вы стали частью самой системы."*

---

## 📖 О проекте

**1984 Bounties Competitive** — это специализированная соревновательная экосистема для игроков osu!, где участники принимают вызовы (баунти) на прохождение карт с заранее установленными условиями.

Система создана для:
- 🎮 Стимулирования игрового прогресса и выхода из зоны комфорта
- 🏆 Формирования здоровой соревновательной и развлекательной экосистемы
- 📊 Отслеживания прогресса через Hunter Progression System (HPS)
- 🎁 Вознаграждения за мастерство, выдержку и лояльность

---

## 🚀 Возможности бота (текущие и дальнейшие)

| Функция | Описание |
|---------|----------|
| 👤 **Профили игроков** | Авто-обновление PP, ранга, страны, точности (каждые 30 мин) |
| 🏆 **Hunter Points** | Система прогрессии с рангами от Candidate до Big Brother |
| 📅 **Сезоны** | 6-месячные циклы с архивацией в Зал Славы |
| 🎯 **Еженедельные баунти** | Автоматические вызовы каждую неделю |
| 📊 **Лидерборды** | Активный рейтинг и Зал Славы (all-time) |
| 🔔 **Уведомления** | Анонсы баунти, напоминания о дедлайнах |

## 📋 Команды бота

### **Основные**
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и информация о боте |
| `/register <nickname>` | Регистрация в системе |
| `/profile` | Показать ваш профиль |
| `/refresh` | Обновить данные из osu! API |
| `/help` | Справка по командам |

### **Рейтинг и сезоны**
| Команда | Описание |
|---------|----------|
| `/leaderboard` | Топ игроков по HP (текущий сезон) |
| `/leaderboard legacy` | Зал Славы (all-time) |
| `/season info` | Информация о текущем сезоне |

### **Баунти**
| Команда | Описание |
|---------|----------|
| `/weekly` | Текущий еженедельный баунти |
| `/bounty list` | Список активных баунти |
| `/submit` | Подать результат на баунти |

## 🏆 Система рангов (HPS)

| Ранг | Диапазон HP | Описание |
|------|-------------|----------|
| 🔵 **Candidate** | 0–250 | Испытуемый. Ваши способности замечены, но лояльность ещё не доказана. |
| 🟢 **Party Member** | 251–750 | Полноправная единица структуры. Вы стабильно функционируете в рамках баунти-циклов. |
| 🟠 **Inspector** | 751–1500 | Опытный субъект. Вы обладаете правом контроля над ритмом и способны навязывать конкуренцию элите. |
| 🔴 **High Commissioner** | 1501–3000 | Высшее руководство. Ваш авторитет незыблем, а рекорды являются эталонными. |
| 🟣 **Big Brother** | 3001+ | Абсолютный предел. Вы больше не участник — вы стали частью самой системы. |

## 🎮 Виды баунти

1. **First FC Bounty** — первый Full Combo на карте
2. **Snipe Bounty** — перебитие рекорда конкретного игрока
3. **History Bounty** — исторически поставленный рекорд
4. **Accuracy Bounty** — рекорд по точности (≥98%, 99%, 100%)
5. **Pass Bounty** — прохождение карты
6. **Mod Bounty** — прохождение с модами (HD/HR/DT/FL и др.)
7. **SS Bounty** — 100% точность
8. **Marathon Bounty** — карты-марафоны (7:00+)
9. **Memory Bounty** — прохождение с Flashlight (FL)
10. **Metronome Bounty** — рекорд по низкому Unstable Rate
11. **Easter Egg Bounty** — ювелирная точность "мемных" цифр

## 🧮 Формула начисления HP

| Результат | Base HP |
|-----------|---------|
| 🥇 Победа | 100 HP |
| ✅ Условие выполнено | 60 HP |
| ⚠️ Частично выполнено | 30 HP |
| 📋 Участие | 10 HP |
| 💰 Спонсорство | 20 HP |

**Множители сложности:**
- 4★ → x0.80
- 5★ → x1.04
- 6★ → x1.28
- 7★ → x1.52
- 8★ → x1.76
- 9★ → x2.00

**Бонусы:**
- 🎯 Flawless Execution (100%): +25 HP
- 🎯 Elite Precision (≥99%): +15 HP
- 🎯 Extra Challenge (HD + HR): +10 HP
- 🎯 Zero Fifty (0x50): +10 HP
- 🎯 Vanguard (первый скор): +15 HP
