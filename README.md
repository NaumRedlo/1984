## 📖 О проекте

**1984 | Global & Competitive** — специализированная экосистема для игроков osu!: аналитика прогресса, красивые карточки статистики, коллекция титулов и собственный GPU-рендер реплеев.

Система создана для:
- 🎮 Стимулирования игрового прогресса и выхода из зоны комфорта;
- 🏆 Формирования здоровой соревновательной и развлекательной экосистемы;
- 🎁 Вознаграждения за мастерство, выдержку и лояльность.

## 📋 Команды бота

Игровые команды намеренно короткие (без длинных синонимов) — набираются без слэша, регистр неважен.

### Основные

| Команда | Описание |
|---------|----------|
| `start` | Приветствие и быстрый старт |
| `register <nickname>` / `reg` | Регистрация в системе |
| `link` | Привязать osu! через OAuth |
| `relink` | Перевыпустить OAuth-токен без потери прогресса |
| `unlink` | Отвязать аккаунт (кулдаун 30 дней) |
| `rf` | Принудительная синхронизация с osu! API |
| `help` | Меню справки |
| `group` / `switch` | (в личке) выбрать группу, с данными которой работает бот |

### osu! статистика

| Команда | Описание |
|---------|----------|
| `pf` | Карточка профиля |
| `rs` | Последняя сыгранная карта, с кнопкой рендера видео |
| `cmp [username]` | Сравнение статистики с другим игроком |
| `lb` / `top` | Таблица лидеров (7 категорий) |
| `lbm [id/url]` | Локальный лидерборд карты |
| `tpp` | Топ-плеи по весовому PP (также доступно кнопкой из `pf`) |
| `tt` | Коллекция титулов |
| `st` | Выбрать активный титул (`st off` — снять) |
| `skin` | Установить скин рендера по ссылке на `.osk` |

### Настройки

| Команда | Описание |
|---------|----------|
| `sts` | Единое меню настроек: рендер (разрешение, HUD, скины), аккаунт, активный титул |

## 🎬 Рендер реплеев

Видео из реплея рендерится на собственном GPU-сервере через [danser](https://github.com/Wieku/danser-go) — без очередей стороннего сервиса.

- **Два входа:** кнопка 🎬 под карточкой `rs` (по последнему скору) или прямая загрузка `.osr`-файла в чат — бот уточнит подтверждением перед рендером, чтобы случайная/спам-загрузка не будила сервер зря.
- **GPU по требованию:** сервер включается только на время рендера и гаснет после — сессии рендера от разных пользователей делят один и тот же запуск, чтобы не платить за лишнее время.
- **Тонкая настройка** через `sts`: разрешение, HUD/курсор/дим, кинематографичный режим (без интерфейса), кастомные скины по ссылке на `.osk`.
- **Библиотека рендеров:** отрисованные видео сохраняются per-игрок, можно переслать повторно без нового рендера или удалить.
- Поддерживается только **osu!standard** — danser не рендерит taiko/catch/mania.

## 🏅 Титулы

Коллекция ачивок за игровые достижения — от «зарегистрируйся в системе» до многочасовых челленджей на топовых картах.

- **7 уровней редкости:** Common → Uncommon → Rare → Epic → Legendary → Mythic → Secret (скрытые условия до разблокировки).
- `tt` показывает прогресс по каждому титулу (в т.ч. незавершённые — сколько осталось), с фильтром по редкости.
- Разблокированный титул можно закрепить как активный (`st`) — он отображается в профиле (`pf`).
- Тексты титулов переведены на русский язык индивидуально под язык карточки игрока.

---

## 📈 Топ-плеи

`tpp` — список лучших скоров аккаунта, посчитанный по той же **системе весового PP**, что и в официальном профиле osu! (место *N* даёт `0.95^(N-1)` от своего PP в общий зачёт).

- Топ-100 (лимит самого osu! API), компактный текстовый список с пагинацией.
- Доступен как отдельной командой, так и кнопкой «🏆 Топ-плеи» под `pf`.
